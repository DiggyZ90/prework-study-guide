Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used first to build the frame to out structure.
  • We use a series of HTML elements to define the content and framework.
  • HTML attributes are always defined in the beginning element, never the closing element

CSS

  • Use CSS to add styling to the content you created using HTML
  • CSS is the language used to describe the presentation of a website
  • that's created with HTML and how each element on a website is rendered
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Three ways to style a webpage using CSS. Inline, internal and external CSS style sheet.
  • When a developer wants to style an element directly in the HTML file, use inline CSS.
  • External CSS style sheets is the preffered way because it allows developers to keep all CSS
  • rules in a seperate file, which makes design changes easier.
  • Declarations contain two improtant components; the CSS property we want to apply, and the value
  • of the property.
  • DRY = Do Not Repeat Yourself.

Git

  • Git is a free,open source version control system the allows web developers
  • to collaborate on projects without over writing eachother's work
  • It allows developers to revert to previous versions of the code if a problem
  • is discovered
  • Everytime we save a version of our work using version control, it is assigned
  • a unique identifier called a SHA, or HASH, that allows us to keep record
  • of the specific changes commited.
  • This unique identifier provides a reference to the specific version of code
  • so we can retrieve it as needed.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a lightweight programming language the is used to provide logic
  • and interactivity for websites.
  • JavaScript uses variables, control flow, conditional logic, array's and for loops
